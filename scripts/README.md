# Workspace Scripts

This directory contains the automation tools for the Yocto development workspace.

## Documentation
For information on how to **use** these tools, see:
- [Tooling Guide](../docs/tooling-guide.md)

For information on how to **develop or maintain** these tools, see:
- [Script Development & Internals](../docs/scripts-development.md)

## Key Files
- `env_init.sh`: Main entry point for the development environment.
- `*.py`: Individual automation scripts aliased in `env_init.sh`.
